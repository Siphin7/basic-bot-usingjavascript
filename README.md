# TeacherBot Introduction

A beginner JavaScript project demonstrating variables, constants, string concatenation, and output generation.

## Project Overview

TeacherBot introduces itself and shares information such as:

- Name
- Nickname
- Location
- Favorite Subject

The project was built as part of learning JavaScript fundamentals.

## Concepts Used

- Variables (`let`)
- Constants (`const`)
- String Concatenation
- JavaScript Functions
- DOM Manipulation
- Browser-Based Output

## Technologies

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/teacherbot-introduction.git
```

2. Open `index.html` in your browser

No installation required.
